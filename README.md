# Лабораторна робота №2: Робота з системою керування версіями Git

## 🎯 Мета роботи
Ознайомлення з основними командами Git, робота з віддаленими репозиторіями на GitHub, створення гілок та виконання операції злиття (Merge) через Pull Request.

## 📋 Що було зроблено:
1. **Ініціалізація проєкту:** Створено репозиторій та завантажено базові файли (`index.html`, `style.css`).
2. **Робота з гілками:** Створено нову гілку `feature-new-task` для розробки нового функціоналу окремо від основної гілки.
3. **Редагування коду:** У новій гілці було додано нові пункти до списку завдань.
4. **Pull Request & Merge:** Виконано запит на злиття змін з гілки `feature-new-task` у гілку `main`.
